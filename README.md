# Expense Tracker

Una aplicación sencilla y eficiente para la gestión de finanzas personales.

## 🚀 Características actuales
- **Registro de transacciones:** Añade ingresos y gastos con descripción y monto.
- **Visualización clara:** Interfaz diseñada para mostrar el balance actual de forma intuitiva.
- **Gestión de historial:** Lista detallada de los movimientos recientes.
- **Persistencia de datos:** Los datos se mantienen guardados localmente para no perder la información al cerrar la aplicación.
